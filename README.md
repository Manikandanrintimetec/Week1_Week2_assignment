# Python Week 1 & 2 Assignments

This repository contains solutions for Week 1 and Week 2 Python assignments.  
The programs demonstrate important Python concepts including:

- Generators  
- Iterators  
- List Comprehension  
- Tuples  
- Object-Oriented Programming (OOP)  
- File Handling  
- Pickle Module  
- NumPy  
- Matplotlib  

All programs take input from the user through the command line.

## Problems Implemented

### Week 1
1. **Generate Fibonacci numbers using a Generator**  
2. **Iterate a given list in reverse direction** using a custom Iterator class  
3. **Print Prime Numbers between 0 to 10** using List Comprehension  
4. **Convert a list into a list of tuples** `(index, value)`  
   - Example: `input = [2,3,4,5,6]` → `output = [(0,2), (1,3), (2,4), (3,5), (4,6)]`

### Week 2
5. **Command line library management application** using Object-Oriented Programming (OOP) and `pickle`:
   - Classes for Book, Student, and Library  
   - Display available books in the library  
   - Lend and receive books from students  
   - Additional functionalities: add/delete books from inventory

6. **Matrix diagonal swap**:
   - Accepts a square matrix of size `MxM`  
   - Swaps elements from left diagonal with right diagonal using NumPy

7. **Text file processing**:
   - Reads a text file and creates a new file with the first alphabet of each word capitalized  
   - Prints the total number of white spaces in the new file

8. **Random numbers frequency analysis using NumPy**:
   - Generate 1000 random integers in the range 1–100  
   - Calculate frequency of unique numbers and plot a histogram  
   - Retain top 5 most frequent values; replace the rest as “Other” in the bar chart

## Python Version
Python 3.11

- `numpy`
